How to Add New Notifications / Circulars

Edit the data.json file in the repository and add a new entry in the following format:

{
  "id": "NT-CGST-2025-001",
  "type": "Notification",
  "tax": "CGST",
  "number": "01/2025-Central Tax",
  "date": "2025-01-15",
  "effectiveDate": "2025-02-01",
  "subject": "Subject of the notification...",
  "category": "Rate / Exemption / Rules / etc.",
  "sections": ["Section XX"],
  "keywords": ["keyword1", "keyword2"],
  "summary": "Brief summary...",
  "fullText": "Full text of the notification..."
}

Steps to contribute on GitHub:
1. Fork the repository
2. Edit data.json and add the new entry
3. Submit a Pull Request
4. Once merged, the site updates automatically via GitHub Pages
